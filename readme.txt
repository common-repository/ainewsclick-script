Бесплатная система рекомендаций новостного контента, увеличивающая вовлеченность пользователей через самообучающийся алгоритм подбора контента.

== Description ==

**Бесплатный виджет с рекомендациями, обменом трафика и монетизацией**

Ainews Click – это бесплатный многофункциональный виджет рекомендаций новостного контента:

* Дает возможность увеличить вовлеченность через рекомендации релевантного контента, тем самым увеличивая время нахождения пользователя на сайте;
* Позволяет монетизировать ресурс с помощью рекламы, которая гармонично вписывается в блок с рекомендациями;
* Дает возможность привлекать новых пользователей на сайт благодаря обмену трафиком между участниками Ainews Click;
* Позволяет адаптировать внешний вид виджета под любой дизайн сайта с помощью конструктора с гибкой настройкой.

**Увеличивайте вовлеченность через систему рекомендаций**

Система использует самообучающийся алгоритм подбора релевантного контента, который постоянно обновляет рекомендации для каждого посетителя.

Язык рекомендаций в виджете соответствует языку публикации, которую просматривает пользователь.

В системе предусмотрена возможность закрепления главных новостей в верхней части виджета. Достаточно задать время размещения и система покажет новость всем пользователям в заданный период времени.

**Привлекайте новую аудиторию благодаря обмену трафика**

Обмен трафиком позволяет привлекать новую аудиторию на свой ресурс. Плюсы системы обмена трафика Ainews Click:

* Релевантное оформление новостей, отсутствие желтых заголовков;
* Модерация продвигаемых в обмене публикаций;
* Отсутствие дублей и публикаций первоисточника на промежуточной странице;
* Только новостной трафик.

Виджет показывает материалы партнеров и благодаря взаимному обмену, на ваш сайт будет приходить аудитория других медиа и СМИ. При создании виджета можно выбрать его тип:

* **Рекомендательный виджет**. Отображает материалы внутри ресурса и рекламу;
* **Виджет обмена трафика**. Показывает только материалы партнеров.

Так, с помощью виджета обмена трафика можно привлечь новую аудиторию, а с помощью рекомендательного виджета удержать и вовлечь текущую.

В раздел «Тизер» можно добавить статьи для показа в виджете других ресурсов. Так ваши материалы увидят пользователи других сайтов и вы получите новый источник трафика для вашего ресурса.

Функция таргетинга в разделе «Тизер» дает возможность показывать контент целевой аудитории, настраивая следующие параметры показа статьи: тип устройства и операционной системы, которые использует целевая аудитория, а также интересующие ее темы. Так, статьи будут показываться лишь той аудитории, для которой контент будет наиболее интересен – это повысит качество привлекаемого трафика.

**Монетизируйте ресурс с помощью рекламы**

Ainews Click позволяет монетизировать ресурс с помощью рекламы, которая гармонично вписывается в блок с рекомендациями. Рекламный контент имеет пометку «Реклама», поэтому пользователь легко отличит его от рекомендаций.

Гибкая настройка рекламных мест позволяет владельцу сайта самому решить на каких местах и в каком количестве будет показана реклама в виджете.

**Кастомизируйте виджет под сайт без потери производительности**

Создание виджета происходит в интерфейсе AinewsClick. Для каждого проекта можно создать неограниченное количество виджетов с помощью конструктора с гибкой настройкой внешнего вида и уникальными шаблонами.

Скрипт виджета оптимизирован, поэтому не окажет влияния на производительность сайта. Загрузка скрипта асинхронная, а все обработчики событий устанавливаются в пассивном режиме.


**Безопасность и модерация**

Каждое рекламное объявление мы проверяем и модерируем на соответствие законодательству и этическим нормам. Функция пометки публикации категории 18+ позволяет виджету учитывать возрастной ценз и рекомендовать только те материалы, которые соответствуют аудитории. Также мы учитываем предпочтения по типам и содержанию рекламы.

**Помощь в интеграции и поддержка**

Поможем с интеграцией для любого типа сайтов (SPA, SSR, SSG и др.) и подскажем где лучше разместить виджет для большего охвата пользователей.
